# Scripts

Папка зарезервирована под автоматизацию сборки.

План:

1. `validate-manifests` - проверить CSV на обязательные поля.
2. `build-client-pack` - собрать клиентский профиль из манифеста.
3. `build-server-pack` - собрать серверный профиль из манифеста.
4. `sync-common-config` - разложить общие конфиги в клиент и сервер.

Пока `.jar` файлы скачиваются и проверяются вручную, чтобы зафиксировать рабочий набор без случайных обновлений.
